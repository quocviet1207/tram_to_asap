# TRAM → ASAP: Motion Retargeting to Humanoid in IsaacGym

A complete step-by-step guide to extract SMPL motions from [TRAM](https://github.com/yufu-wang/tram) and retarget them to a humanoid robot (Unitree G1) using [ASAP](https://github.com/LeCAR-Lab/ASAP) for training in IsaacGym.

---

## Overview

```
TRAM (video → SMPL)
        ↓
  hps_track_0.npy + camera.npy
        ↓
  convert_tram.py   ← (YOU WRITE THIS)
        ↓
  IMG_7483_track0.npz   (ASAP SMPL format)
        ↓
  fit_smpl_shape.py     (already in ASAP)
        ↓
  fit_smpl_motion.py    (already in ASAP)
        ↓
  0-motions_raw_..._IMG_7483_track0.pkl   (G1 DOF motion)
        ↓
  train_agent.py        (IsaacGym RL training)
```

---

## Prerequisites

- TRAM installed and run on a video — you need the output at `results/<video_name>/`

## Understanding the TRAM Output Keys

When you load `hps_track_0.npy` you get these actual keys (not what the docs say):

```python
hps = np.load('hps_track_0.npy', allow_pickle=True).item()
# dict_keys(['pred_cam', 'pred_pose', 'pred_shape', 'pred_rotmat', 'pred_trans', 'frame'])

# pred_pose:    torch.Size([T, 144])   — 6D rotation (24 joints × 6), NOT axis-angle
# pred_shape:   torch.Size([T, 10])    — SMPL betas per frame
# pred_rotmat:  torch.Size([T, 24, 3, 3])  — rotation matrices (use this, not pred_pose)
# pred_trans:   torch.Size([T, 1, 3])  — camera-space translation (needs .squeeze(1))
# pred_cam:     torch.Size([T, 3])     — weak-perspective cam params
# frame:        torch.Size([T])        — frame indices
```

```python
camera = np.load('camera.npy', allow_pickle=True).item()
# dict_keys(['pred_cam_R', 'pred_cam_T', 'world_cam_R', 'world_cam_T', 'img_focal', 'img_center', 'spec_focal'])

# world_cam_R:  (T, 3, 3) — use this (NOT 'R')
# world_cam_T:  (T, 3)    — use this (NOT 'T')
```

The ASAP `.npz` motion format requires these exact keys/types:

```
trans:           shape=(T, 3),  dtype=float32
gender:          shape=(),      dtype=<U7   (unicode string)
mocap_framerate: shape=(),      dtype=int64
betas:           shape=(16,),   dtype=float64   ← 16 not 10! pad with zeros
poses:           shape=(T, 72), dtype=float32
```

## Phase 3 — Convert TRAM Output to ASAP Format

### 3.1 Create the output folder

```bash
mkdir -p ~/Documents/ASAP/humanoidverse/data/motions/raw_tairantestbed_smpl/
```

### 3.2 Create `~/Documents/ASAP/convert_tram.py`

```python
# ~/Documents/ASAP/convert_tram.py
import numpy as np
from scipy.spatial.transform import Rotation as R

def to_np(x):
    return x.numpy() if hasattr(x, 'numpy') else np.array(x)

def tram_to_asap_npz(hps_path, camera_path, output_path):
    hps    = np.load(hps_path,    allow_pickle=True).item()
    camera = np.load(camera_path, allow_pickle=True).item()

    rotmat     = to_np(hps['pred_rotmat'])            # (T, 24, 3, 3)
    betas      = to_np(hps['pred_shape'])             # (T, 10)
    pred_trans = to_np(hps['pred_trans']).squeeze(1)  # (T, 3)  ← squeeze dim 1
    R_world    = to_np(camera['world_cam_R'])          # (T, 3, 3)
    T_world    = to_np(camera['world_cam_T'])          # (T, 3)
    T          = rotmat.shape[0]

    # Camera space → world space translation
    cam_p       = pred_trans - T_world
    world_trans = (R_world.transpose(0,2,1) @ cam_p[..., np.newaxis]).squeeze(-1)
    world_trans[:, 2] -= world_trans[:, 2].min()      # floor snap (Z=0)

    # Rotation matrices → axis-angle (T, 72)
    pose_aa = R.from_matrix(
        rotmat.reshape(-1, 3, 3)
    ).as_rotvec().reshape(T, 72).astype(np.float32)

    # ASAP uses betas shape (16,) float64 — pad our (10,) with zeros
    betas_16 = np.zeros(16, dtype=np.float64)
    betas_16[:10] = betas.mean(axis=0).astype(np.float64)

    save_dict = {
        'poses':           pose_aa,                             # (T, 72)  float32
        'trans':           world_trans.astype(np.float32),      # (T, 3)   float32
        'betas':           betas_16,                            # (16,)    float64
        'gender':          np.str_('neutral'),                  # scalar   unicode
        'mocap_framerate': np.int64(30),                        # scalar   int64
    }

    np.savez(output_path, **save_dict)
    print(f"\n✅ Saved {T} frames → {output_path}.npz")

    # Verify it matches ASAP reference format
    print("\n🔍 Verifying saved file:")
    check = np.load(output_path + '.npz', allow_pickle=True)
    for k in ['trans', 'gender', 'mocap_framerate', 'betas', 'poses']:
        v = check[k]
        print(f"  {k}: shape={v.shape}, dtype={v.dtype}")


tram_to_asap_npz(
    hps_path    = '/home/viet/Documents/tram/results/IMG_7483/hps/hps_track_0.npy',
    camera_path = '/home/viet/Documents/tram/results/IMG_7483/camera.npy',
    output_path = 'humanoidverse/data/motions/raw_tairantestbed_smpl/IMG_7483_track0'
)
```

## Phase 6 — Fix IsaacGym `libpython` Error

Before training, you must set the `LD_LIBRARY_PATH` or IsaacGym will fail with:
```
ImportError: libpython3.8.so.1.0: cannot open shared object file
```

Fix (add to `~/.bashrc` to make permanent):
```bash
export LD_LIBRARY_PATH=/home/viet/anaconda3/envs/hvgym/lib:$LD_LIBRARY_PATH
echo 'export LD_LIBRARY_PATH=/home/viet/anaconda3/envs/hvgym/lib:$LD_LIBRARY_PATH' >> ~/.bashrc
source ~/.bashrc
```

## Common Errors and Fixes

| Error | Cause | Fix |
|-------|-------|-----|
| `ImportError: libpython3.8.so.1.0` | IsaacGym can't find conda's libpython | `export LD_LIBRARY_PATH=/home/viet/anaconda3/envs/hvgym/lib:$LD_LIBRARY_PATH` |
| `KeyError: 'pose'` | Wrong hps.npy key names | Use `pred_pose`, `pred_rotmat`, `pred_trans`, `pred_shape` |
| `KeyError: 'R'` | Wrong camera.npy key names | Use `world_cam_R` and `world_cam_T` |
| Motion file not found at `0-IMG_7483...` | ASAP prepends full path to filename | Use full generated name: `0-motions_raw_tairantestbed_smpl_IMG_7483_track0.pkl` |
| `pred_trans` shape errors | Shape is `(T, 1, 3)` not `(T, 3)` | Call `.squeeze(1)` after loading |
| `betas` shape mismatch | ASAP expects `(16,)` float64, TRAM gives `(10,)` | Pad with zeros: `betas_16 = np.zeros(16); betas_16[:10] = betas_mean` |

## Notes

- `pred_pose` in TRAM is **6D rotation** `(T, 144)` — do **not** use it as axis-angle. Always use `pred_rotmat` and convert via `scipy.spatial.transform.Rotation`.
- `pred_trans` is in **camera space** — transform to world space using `world_cam_R` and `world_cam_T` from `camera.npy`.
- ASAP's retargeted motion filename is auto-generated as `0-motions_raw_tairantestbed_smpl_<your_input_name>.pkl`.
- Shape fitting (`fit_smpl_shape.py`) only needs to be run once per robot type, not once per motion.
